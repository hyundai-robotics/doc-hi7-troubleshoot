# E02281. 서보ON 상태에서 전자접촉기(MC1) 고장/검지이상

## 1. 개요

서보 ON 상태에서 마그네트(전자접촉기, magnetic contactor) MC1이 비정상적으로 OFF되었습니다.

## 2. 원인 및 점검 방법

{% hint style="info" %}

(1)	모니터링 계통을 점검하십시오.<br>
(2)	마그네트 MC1를 점검하십시오.<br>
(3)	전장보드를 점검하십시오.<br>
(4)	서보안전 보드를 점검하십시오.<br>

{% endhint %}

세부적인 점검방법은 “E02280 서보ON 시도중 전자접촉기(MC1) 고장/검지 이상”을 참조하십시오.
