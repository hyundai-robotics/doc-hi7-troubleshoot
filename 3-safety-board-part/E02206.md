# E02206. 본체 리밋SW 단선 또는 미접속

## 1. 개요

로봇이 소프트 리밋 영역을 이탈하지 않았음에도 로봇 각 축의 동작 영역 끝에 설치한 리밋스위치가 작동한 것으로입력되고 있습니다. 비정상적인 상황이므로 리밋SW배선을 확인해야 합니다.

## 2. 원인 및 점검 방법

{% hint style="info" %}

소프트 리밋을 넘지 않은 상황에서 하드웨어 리밋스위치 작동이 검지되는 비정상적인 상황입니다.

스위치나 배선계통에 문제가 있으므로 점검하십시오.

{% endhint %}

E02206 본체 리밋SW 단선 또는 미접속 발생

소프트 리밋을 넘지 않은 상황에서 하드웨어 리밋SW 작동이 검지되는 비정상적인 상황입니다. 스위치나 배선계통에 문제가 있으므로 점검하십시오.

* 하드웨어 리밋SW 고장: 스위치의 파손이나 어떠한 이유에서 개방(Open)됨.
* 배선: 배선이 끊겼거나 손상을 입어 접촉불량 발생.
* 커넥터: 커넥터가 빠졌거나 손상을 입어 접속불량 등의 단선문제 발생.

상세한 점검 포인트는 “하드웨어 리밋 스위치 점검방법” 항을 참조하십시오.
