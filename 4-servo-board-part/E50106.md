# E50106. (O축) 엔코더 데이터 비정상 변화 감지

## 1. 개요

서보 안전 보드는 서보 모터 제어를 위해 엔코더와 시리얼 통신을 수행하며, 주기적으로 엔코더 데이터를 수신합니다.<br>
해당 에러는 엔코더로부터 수신된 위치데이터가 비정상적으로 변동하는 경우 발생합니다.<br>

엔코더로부터 수신된 위치데이터가 비정상적으로 변동하는 경우는 주로 배선 불량 또는 엔코더 쉴드 라인 처리 문제로 인해 엔코더 신호선에 노이즈가 유입될 때 발생할 수 있습니다.


## 2. 원인 및 점검

{% hint style="info" %}

(1)	엔코더 배선을 점검하십시오.<br>
(2)	모터(엔코더)를 교체 시험하십시오.<br>
(3)	서보보드를 교체 시험하십시오.<br>
(4)	조치 완료후 배선의 통신상태를 점검하십시오.<br>

{% endhint %}

세부적인 점검방법은 "E50101(O축) 엔코더 명령 필드 설정 이상"을 참조하십시오.
