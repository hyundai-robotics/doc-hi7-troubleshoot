# E50107. (O축) 엔코더 단선(모터 ON)

## 1. 개요

서보 안전 보드는 서보 모터 제어를 위해 엔코더와 시리얼 통신을 수행하며, 주기적으로 엔코더 데이터를 수신합니다.<br>
해당 에러는 엔코더와 서보 안전 보드 간의 통신이 불안정하거나 신호가 전혀 수신되지 않을 경우 발생합니다.<br>

엔코더와 서보 안전 보드 간의 통신이 불안정하거나 신호가 수신되지 않는 경우는 주로 엔코더 케이블의 단선 또는 접촉 불량으로 인해 발생할 수 있습니다.


## 2. 원인 및 점검

{% hint style="info" %}

(1)	엔코더 공급전압을 확인하십시오.<br>
(2)	엔코더 배선을 점검하십시오.<br>
(3)	서보보드를 교체 시험하십시오.<br>
(4)	모터(엔코더)를 교체 시험하십시오.<br>
(5)	조치 완료후 배선의 통신상태를 점검하십시오.<br>

{% endhint %}

세부적인 점검방법은 "E50101(O축) 엔코더 명령 필드 설정 이상"을 참조하십시오.
