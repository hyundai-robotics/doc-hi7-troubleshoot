# E50110. (O축) 엔코더 위치 정보 초기화 이상 (FS 비트 감지)

## 1. 개요

서보 안전 보드는 서보 모터 제어를 위해 엔코더와 시리얼 통신을 수행하며, 주기적으로 엔코더 데이터를 수신합니다.<br>
해당 에러는 엔코더 내부에서 Full Absolute 상태에 도달하지 못해 정확한 위치 정보가 아직 확보되지 않은 경우 발생합니다. 이는 엔코더 전원 인가 직후, 데이터 초기화 실패 또는 백업 정보 손실로 인해 발생할 수 있습니다.<br>
본 에러는 엔코더로부터 수신된 데이터 자체는 정상이나, 엔코더 내부 상태 감시 결과 에러 상태(FS)로 판단된 경우에 발생할 수 있습니다.<br>

**FS(Full Absolute)** : 해당 비트가 0인 경우 위치 데이터가 초기화되지 않았거나 백업 정보가 불완전한 상태를 의미하며,
해당 비트가 1인 경우 엔코더 내부 데이터가 완전하고 정상 상태임을 나타냅니다.


## 2. 원인 및 점검

{% hint style="info" %}

(1)	엔코더 공급전압을 확인하십시오.<br>
(2)	시리얼 엔코더 에러해제 후, 제어기 전원을 껏다카 다시 켜십시오.<br>
(3)	계속 에러발생시 모터(엔코더)를 교체 시험하십시오.<br>

{% endhint %}

세부적인 점검방법은 "E50108 (O축) 엔코더 내부 회전값 오류(CE 비트 감지)"을 참조하십시오.
