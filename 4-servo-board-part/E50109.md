# E50109. (O축) 엔코더 과속 오류 (OS 비트 감지)

## 1. 개요

서보 안전 보드는 서보 모터 제어를 위해 엔코더와 시리얼 통신을 수행하며, 주기적으로 엔코더 데이터를 수신합니다.<br>
해당 에러는 엔코더 회전 속도가 허용 범위를 초과하여 OS(OverSpeed) 비트가 세트되었을 때 발생합니다. 이는 실제 과속 상태이거나, 신호 이상으로 인해 과속으로 오검출된 경우일 수 있습니다.<br>

본 에러는 엔코더로부터 수신된 데이터 자체는 정상이나, 엔코더 내부 상태 감시 결과 에러 상태(OS)로 판단된 경우에 발생할 수 있습니다.

## 2. 원인 및 점검

{% hint style="info" %}

(1)	엔코더 공급전압을 확인하십시오.<br>
(2)	엔코더 배선을 점검하십시오.<br>
(3)	시리얼 엔코더 에러해제 후, 제어기 전원을 껏다카 다시 켜십시오.<br>
(4)	계속 에러발생시 모터(엔코더)를 교체 시험하십시오. 

{% endhint %}

세부적인 점검방법은 "E50108 (O축) 엔코더 내부 회전값 오류(CE 비트 감지)"을 참조하십시오.
