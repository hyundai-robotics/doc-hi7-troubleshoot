# E02541 구동장치 제어전압 저하

## 1. 개요

서보 구동장치에 공급되는 제어전원인 +15V가 저하되었습니다. 해당 에러는 제어기별로 다른 경로에서 검지하여 서보 보드로 전달됩니다.

*   Hi6-N : 서보 구동장치에서 검지
*   Hi6-T : 백플레인보드(BD602T)에서 검지

## 2. 원인 및 점검방법

{% hint style="info" %}

*   <전원 LED 확인>

    (1)	전원 표시 LED를 확인하여 주십시오.

    ->  Hi7-N : 서보 구동장치의 ‘POW’ LED를 확인하여 주십시오.

    ->  Hi7-T : To be included in the future
    
    ->  Hi7-NX : To be included in the future

    ->  제어전원 공급장치의 PSM에서 제어전원 24V 전압을 확인하여 주십시오.


*   <보드 LED와 PSM LED가 모두 OFF된 경우>

    (2)	제어전원 공급장치의 출력을 확인하여 주십시오.

    ->  Hi7-N

    *  Control Module의 BD604에 연결된 CN24VB1 커넥터 연결을 분리한 후 PSM의 24V 출력확인용 LED를 확인하여 주십시오.

    *   BD642 보드를 분리한 후 서보구동장치의 ‘POW’ LED를 확인하여 주십시오. 

    ->  Hi7-T : To be included in the future

    ->  Hi7-NX : To be included in the future

    (3)	제어전원 공급장치를 점검하여 주십시오.

    ->  CMSMPS에 입력되는 전압을 확인하여 주십시오.

    ->  CMSMPS을 교체한 후 LED를 확인하여 주십시오.

    * <보드 LED만 OFF된 경우>

    (4)	관련 부품을 교체하여 전원 표시 LED를 확인하십시오.

    -> Hi7-N

    * PSM과 Control Module을 연결하는 CN24VB1 케이블을 교체한 후 LED를 확인하여 주십시오.

    * 서보 보드를 교체한 후 LED를 확인하여 주십시오.

    * 서보 구동장치를 교체한 후 LED를 확인하여 주십시오.

    ->  Hi7-T : To be included in the future
